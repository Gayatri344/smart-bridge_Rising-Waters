# 🌊 About the Project

Floods are one of the most dangerous natural disasters, causing damage to people, property, and the environment.

**Rising Waters** is a Machine Learning-based web application that predicts the possibility of floods using rainfall and weather data. The system helps provide early warnings, enabling people and authorities to take preventive measures before flooding occurs.
